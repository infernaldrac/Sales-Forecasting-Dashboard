# 📈 Sales Forecasting and Business Intelligence Dashboard

## Overview

This project was developed as part of the AI & Data Science Internship.

The objective is to analyze historical Superstore sales data, forecast future sales using multiple forecasting techniques, compare forecasting models, and build an interactive Streamlit dashboard for business decision-making.

---

## Features

- Data Cleaning & Preprocessing
- Exploratory Data Analysis
- Time Series Analysis
- Holt-Winters Forecasting
- SARIMA Forecasting
- Prophet Forecasting
- XGBoost Forecasting
- Model Performance Comparison
- Future Sales Forecast
- Category & Region Forecasting
- Product Demand Segmentation
- Anomaly Detection
- Interactive Streamlit Dashboard

---

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Plotly
- Scikit-Learn
- Statsmodels
- Prophet
- XGBoost
- Streamlit
